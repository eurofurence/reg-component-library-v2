# Form

The `Form` component should be used to contain form controls.
It defines a vertical grid of 10 columns upon which the form controls can be placed.

The following components adhere to this grid when used as direct children to the `Form` component:

- `TextField`
- `Checkbox` (labeled)
- `RadioItem` (labeled)
- `RadioSet`
- `FieldSet`

These components also accept a `gridSpan` prop, which can be used to define the amount of columns
the fields use when not being collapsed by responsive media queries.
